Blockchain que es? No es BlockChange
Conceptos
  • Peer: miembro de la red, identificable y verificable.
  • Ledger: libro contable, base de datos compartida, replicada y sincronizada entre los miembros.
  • Asset: activo tangible ( auto, casa,etc ) como intangible (un patente, contrato , hipoteca, etc.)
  • Transacciones: es una transferencia de un Asset dentro o fuera del ledger.
  • Consenso: proceso de colaboración entre miembros para acordar cuando una transacción es válida.
  • Smart contracts / Chaincode: codigo almacenado en la red, que gobierna todas las interacciones con el ledger.
  • Canales: una subred entre 2 o mas miembros de una red

    • Caractéristicas
  • Colaborativa: entre los nodos participantes
  • Distribuida :(peer to peer) cada nodo tiene una copia del ledger.
  • Descentralizada: no hay un solo servidor
  • Inmutable: Solo de agregado de información.
    • Tipos
  • Publicas: de acceso general. Bitcoin, Ethereum, Cardano, Litecoin
  • Privadas : solo para integrantes del consorcion. Hyperledger Fabric, Quorum , Corda
    • Redes Blockchain Privadas
  • Consorcio
  • Permisionadas ( de pertenencia, de operacion )
  • Consenso relajado
  • Privacidad: Assets y tx solo visibles para los participantes
